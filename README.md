# ⏱️ Task 2 - Animated Stopwatch Web Application

## 📌 Project Overview
This project is an interactive and responsive Stopwatch Web Application developed using HTML, CSS, and JavaScript. The application allows users to track time accurately with millisecond precision and includes essential stopwatch features such as Start, Pause, Reset, and Lap recording.

## 🚀 Features
- Start, Pause, and Reset functionality
- Lap time recording
- Real-time millisecond tracking
- Animated button effects and hover interactions
- Running glow animation
- Scrollable lap list
- Responsive and modern UI design

## 🛠️ Technologies Used
- HTML5
- CSS3 (Animations & Transitions)
- JavaScript (DOM Manipulation & Time Functions)

## 🎯 Learning Outcomes
- Understanding JavaScript timing functions (setInterval, Date)
- DOM manipulation and dynamic updates
- Event handling in JavaScript
- Creating responsive layouts
- Implementing CSS animations and UI effects
